#### 1. 下列语法正确的是（ABC)

    B. while-else
    C. for-else

> 相关知识点： 对于while ... else ...、try ... except ... else ... 、for ... else ...来说，当语句体中有break、return或者异常发生则不会执行else语句体；其余情况下()则会执行else语句体.
